# LLD (Low-Level Design) Learning Journey

Welcome to my Low-Level Design (LLD) learning repository!  
This repository contains hands-on Java code examples and explanations for core Object-Oriented Programming (OOP) concepts, UML diagrams, SOLID principles, and various design patterns.  
Each folder focuses on a specific topic or pattern, with real-world analogies and code to help reinforce understanding.

---

## Folder Structure

- **00. OOPS_CONCEPT/**  
  Fundamental OOP concepts with practical car-based examples:
  - [`Abstraction.java`](00.%20OOPS_CONCEPT/Abstraction.java):  
    Demonstrates abstraction using interfaces and concrete classes.
  - [`Encapsulation.java`](00.%20OOPS_CONCEPT/Encapsulation.java):  
    Shows encapsulation with private fields, getters, setters, and data security.
  - [`Inheritance.java`](00.%20OOPS_CONCEPT/Inheritance.java):  
    Explains inheritance with parent and child classes, code reuse, and specialization.
  - [`StaticPolymorphism.java`](00.%20OOPS_CONCEPT/StaticPolymorphism.java):  
    Illustrates static (compile-time) polymorphism via method overloading.
  - [`DynamicPolymorphism.java`](00.%20OOPS_CONCEPT/DynamicPolymorphism.java):  
    Demonstrates dynamic (runtime) polymorphism using abstract classes and method overriding.
  - [`StaticAndDynamicPolymorphism.java`](00.%20OOPS_CONCEPT/StaticAndDynamicPolymorphism.java):  
    Combines both static and dynamic polymorphism in a single example.

- Browse each folder for focused examples and explanations.
